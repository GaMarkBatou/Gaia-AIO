Ticketuebersicht Customer Buttons

id='null' - You can put here tt number but there is no point, leave it.
type='button' - Do not touch!
class='toprightnow' - Do not touch!
data-tstatus='alle Offenen, nicht behoben' - Ticket Status
data-group='TCS.DF.HU.OPS.L1.SDWAN.SDX' Gruppen
data-kname='' Konzernname + eg.:Georg Fischer Group
data-main='main' main - Main-TTs | null aka '' - Sub-TTs
data-suchk='' Suchkriterien - [eg.: Incidentklasse = "% torung %" ]
title=''>Null</button> Instead of "null" write it the button title.

Put the code next to each other closely [...</button><button...] or you can use separate character [...</button> | <button...] to your taste.



Kundenstatus Templates


Start with "["
KS start with - {
Text is the "title" of the ks - "text": "----Erstmeldung--1--",
The value is the full KS - "value": "Dear Sir or Madam,\n\nWe have ....\n\nBest regards\nDeutsche Telekom\nService Desk"
use "\n" when you break the line.
KS end with - }
between two ks use ","
End with "]"
